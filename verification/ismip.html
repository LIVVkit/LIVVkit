LIVV: Land Ice Verification & Validation Documentation

ismip

The Ice Sheet Model Intercomparison Project for Higher-Order Models (ISMIP-HOM) prescribes a set of experiments meant to test the implementation of higher-order physics. For more information, see http://homepages.ulb.ac.be/~fpattyn/ismip/ Simulates steady ice flow over a surface with periodic boundary conditions
